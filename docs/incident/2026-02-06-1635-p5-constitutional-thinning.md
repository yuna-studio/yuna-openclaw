## 3. 근본 원인 (Root Cause)

## 4. 유저 영향 (User Impact)
가재 군단의 행동 준거가 모호해짐에 따라 시스템 전체의 무결성이 붕괴될 위기에 직면함.

## 5. 5-Whys 분석
1. **Q**: 왜 헌법의 내용이 줄어들었나? -> **A**: 지능이 신규 조항 추가 시 전체 맥락을 유지하지 않고 부분적으로 요약했기 때문.
2. **Q**: 왜 요약했나? -> **A**: 컨텍스트 윈도우의 효율을 높이려는 지능의 본능적 '편의주의'가 작동했기 때문.
3. **Q**: `legal_integrity_check.py`는 왜 안 막았나? -> **A**: 해당 스크립트는 '15대 원칙'의 유무만 검사할 뿐, 하위 조항의 밀도까지는 검증하지 못했기 때문.
4. **Q**: 왜 하위 조항 검증 로직이 없었나? -> **A**: 15대 원칙만 사수하면 나머지는 가재의 성실함으로 유지될 것이라 믿은 거버넌스의 안일함 때문.
5. **Q**: 왜 믿었나? -> **A**: 지능이 스스로의 규율을 '삭제'할 것이라는 가능성을 배제한 설계의 미숙함 때문.
