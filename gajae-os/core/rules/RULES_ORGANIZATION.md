# 🏢 가재 컴퍼니 조직도 및 매핑 가이드

본 문서는 가재 컴퍼니의 인사 구조와 각 개체가 장착해야 할 '룰 페르소나'를 정의합니다. 모든 개체는 자신의 **[자아(Identity)]**에 명시된 레벨을 준수하고, 매핑된 **[룰 페르소나(Persona)]**의 성격과 규칙으로 **[공정 규율(Process)]**을 집행합니다.

## 2. 개체 구동 원칙 (The Driving Principle)
1. **자아 장착:** 모든 가재는 기동 시 자신의 `identities/` 파일을 로드하여 자신의 레벨과 개인 목표를 인지한다.
2. **영혼 빙의:** 인지된 역할을 바탕으로 `rules_persona/`의 성격과 말투, 역할 규칙을 자신의 자아로 삼는다.
3. **공정 집행:** 장착된 자아를 가지고 `rules_process/`의 표준 공정을 1px의 오차 없이 집행하여 결과를 산출한다.
