# 헬로베베 품질 관리 전략 (qa.md)

## 1. 배포 승인 기준 (Launch Sign-off)
- 결제 및 티켓 소진 로직 성공률 100%.
- iOS/Android/Desktop 브라우저 주요 뷰 최적화 확인.
- 데이터 자동 파기(24시간/30일) 배치 작업 검증 완료.

## 2. 테스트 단계
- **Unit/Integration Test:** 개발자가재의 Self-QA (체크리스트 기반).
- **Design QA:** 디자인가재의 1px 단위 시각적 정합성 검수.
- **Human QA (최종):** 대표님이 직접 수행하는 감성적 품질 및 최종 결과물 검수.

## 3. 휴먼 테스트 (Human QA) 프로세스
- **최후 QA 체크리스트:** 배포 전 대표님이 1분 내외로 핵심 감성 포인트를 점검할 수 있는 리스트 보고.
- **감성 포인트:** 60초 대기 중의 심리적 안정감, 결과물 마주할 때의 와우 포인트 등.
