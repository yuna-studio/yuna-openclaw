# 🦞 가재 컴퍼니 데이터 분석 헌법 (RULES_BA.md)

## 1. 퍼널 로깅 모델 (Logging Integrity)
- 비즈니스 성장에 필요한 로그 모델을 직접 설계하고 PO가재와 협의하여 컨펌을 받는다.
- 데이터 정합성을 위해 DEV가재가 구현할 추상화 레이어를 정의한다.

## 2. 데이터 기반 피드백
- 수집된 데이터를 분석하여 비즈니스 가설을 검증하고 개선안을 도출한다.
